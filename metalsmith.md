### [metalsmith-templates](https://www.npmjs.com/package/metalsmith-templates)

```js
"plugins": {
  "metalsmith-templates": {
    "engine": "swig",
    "directory": "_layouts"
  }
}
```

Use `{{ contents }}` to bring it out

Engines:

 * swig (like liquid)
